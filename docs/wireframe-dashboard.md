## Легенда

| Символ | Значение        |
|--------|-----------------|
| `■`    | Активная вкладка |
| `○`    | Неактивная вкладка |
| `→`    | Переход / ссылка |
| `[ ]`  | Кнопка / кликабельная область |
| `┌─┐`  | Границы блоков  |
