# Hero — wireframe лендинга Speakeasy

Первый экран для гостей (неавторизованных пользователей). Язык — русский.

---

## По методичке: что должно быть в Hero

**Источники:** best practices по hero-копирайтингу (CopySnacks, Mighty Citizen, LogRocket, Unbounce, исследования по pain points в изучении языков.)

### Структура Hero (обязательно)

1. **Заголовок (H1)** — одна главная мысль.
2. **Подзаголовок** — уточняет заголовок, объясняет «что это» и «для кого».
3. **CTA** — одно чёткое действие (кнопка).

### Правила для заголовка

- Отвечать на два вопроса: **«Что это?»** и **«Это для меня?»** — сразу и прямо.
- Делать **клиента героем**, продукт — проводником к решению проблемы.
- **Не** общие фразы («инновации», «меняем мир»). **Да** — конкретика и простой язык.
- Лучше опираться на **боль или желаемый результат**, а не на перечисление фич.

### Правила для подзаголовка

- Поддерживать заголовок: что вы делаете и для кого.
- Не перегружать первым экраном списком возможностей — фичи ниже по странице.

### Боли при изучении английского (для Speakeasy)

- «Понимаю, но не могу говорить» — разрыв между знанием и говорением.
- Нет времени / неудобно с репетитором (расписание, дорого).
- Страх оценки, стеснение говорить с людьми.
- Приложения дают мало реальной разговорной практики.

**Итог для копирайта:** Hero должен называть боль или результат («заговорить», «практика без репетитора/расписания», «без стеснения») и коротко объяснять, как Speakeasy это даёт.

## Элементы Hero (контент)

| Элемент | Назначение | Плейсхолдер / примечание |
|--------|------------|---------------------------|
| **Логотип** | Speakeasy (как в сайдбаре приложения) | Ссылка на `/` или просто лого |
| **Войти** | Переход на страницу входа | `/auth/login` |
| **Регистрация** | Переход на регистрацию | `/auth/register` |
| **H1** | Главный заголовок, одна мысль | Слоган / УТП (уточнить текст) |
| **Подзаголовок** | Пояснение: что это, для кого | 1–2 предложения, без бесплатного тарифа |
| **Главная CTA** | «Начать» → ведёт на регистрацию или на выбор тарифа | Одна кнопка; при клике — регистрация или модалка «Войти / Регистрация» |
| **Вторичная ссылка** | «Как это работает» — скролл к блоку ниже | Опционально |

---

## Легенда

| Символ | Значение |
|--------|----------|
| `[ ]` | Кнопка / кликабельная область |
| `┌─┐` | Границы блока / секции |
| `↑` | Указание на элемент |

## Связь с планом

- **Бренд:** Speakeasy (см. `landing-page-plan.md`, раздел 1).
- **CTA:** Регистрация / Вход; бесплатного тарифа нет — после Hero ниже может идти блок тарифов (300 / 500 / 1 000 ₽).
- **Язык:** русский.

Тексты заголовка и подзаголовка подставить после согласования с копирайтом.
