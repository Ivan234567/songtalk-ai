## 9) Recent Technical Decisions (Auth + AI Access)

- Зафиксирована причина ошибки `Invalid or expired token`: фронтенд в ряде модулей использовал `backend_jwt` из `localStorage` без проверки срока жизни (`exp`), из-за чего AI-функции могли падать при уже пополненном балансе.
- Принято решение унифицировать работу с backend JWT через единый helper: `frontend/lib/backend-jwt.ts`.
- Что теперь делает helper:
  - валидирует `exp` в токене;
  - удаляет просроченный токен из `localStorage`;
  - дает единые методы `get/store/clear` для всех модулей.
- На helper переведены ключевые AI-модули: `AgentTab`, `TranslatorPanel`, `DebateSetupUI`, `VideosPage`, `user-scenarios`.
- При `signOut` backend JWT теперь гарантированно очищается (dashboard и landing), чтобы не оставлять "висячие" токены после выхода.
- Практический эффект: баланс и доступ к AI-функциям снова синхронизированы; снижен риск ложных отказов доступа из-за протухшего локального токена.
